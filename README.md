# Del-Cuello-de-Botella-a-la-Concurrencia-Multihilo
Demostrar de forma práctica las limitaciones de rendimiento de un servidor secuencial (iterativo) de red, y aplicar el modelo multihilo (un servidor por cliente) para diseñar y programar un servidor TCP concurrente capaz de manejar múltiples solicitudes simultáneamente sin bloquearse.
